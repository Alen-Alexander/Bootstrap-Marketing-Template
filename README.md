# Bootstrap-Marketing-Template
Complete Bootstrap 4 marketing website template with responsive navbar, sidebar, hero section, and content grid. Production-ready landing page starter.

Features
- **Responsive Navbar**: Collapsible mobile menu + user status
- **Sidebar Navigation**: 3-section menu layout (`col-sm-3`)
- **Hero Jumbotron**: Full-width call-to-action section
- **Content Grid**: 3x4 responsive cards (`col-sm-4`)
- **Interactive Modal**: Centered popup with Bootstrap components

✅ Full-page responsive layouts
✅ Bootstrap navbar + grid mastery
✅ Component integration (modal + jumbotron)
✅ Mobile-first responsive design
